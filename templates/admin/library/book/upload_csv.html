How to Prepare Your CSV Data

Please make sure your CSV file has the following columns. Fields marked with an asterisk (*) are required.

Columns:

  • title*: The main title of the book.
  • subtitle: The subtitle, if any.
  • author*: The name(s) of the author(s). For multiple authors, separate them with a semicolon (e.g., "Author One;Author Two").
  • publisher*: The name of the publisher.
  • publication_year*: The year the book was published (e.g., 2023).
  • isbn_10: The 10-digit ISBN.
  • isbn_13: The 13-digit ISBN.
  • classification_number*: The Dewey Decimal Classification number (e.g., 230.1).
  • cutter_number*: The Cutter number (e.g., S64i).
  • category*: The category or subject of the book (e.g., Systematic Theology).
  • language: The language code (e.g., en, bn, hi).
  • pages: The number of pages.
  • edition: The edition of the book (e.g., 3rd Edition).
  • description: A brief description of the book.
  • keywords: Comma-separated keywords.
  • total_copies: The total number of copies.
  • copies_available: The number of copies currently available.
  • location_shelf: The physical location in the library (e.g., A-1-5).
  • status: The availability status (e.g., available, checked_out).
